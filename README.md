# tryToUseBlueTooth
一个使用CoreBlueTooth编写的扫描附近蓝牙设备（中心模式），显示在列表上并进行连接的程序.

##技术要点
1. CoreBlueTooth包含的几个协议方法的使用与理解
2. 多线程CenralManager的使用和数据回调，中间的逻辑先后与处理方法。
3. 使用代理，通知与tableView进行数据上的交互
